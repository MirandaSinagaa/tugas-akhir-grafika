GRAFIKA 2025: Pengumpulan video demo program
Buatlan program yang bisa membaca dua file yang masing-masing berisi informasi titik-titik koordinat 3D (x,y,z) dan satu file lain berisi informasi pasangan-pasangan titik yang terhubung.
Program membaca dua file tersebut, dan menampilkannya ke layar menggunakan manipulasi pixel langsung. Penggunaan library plotting seperti matplotlib tidak diperbolehkan.
Sertakan juga fungsi atau fitur untuk memanipulasi titik-titik dan garis tersebut dengan tranformasi Skala, Translasi, dan Rotasi.
